문제 6. 아스키코드를 담는다고 했을 때 VARCHAR(65535) 선언이 가능하다. (O / X)

**정답** X
**해설** 아스키코드를 담는다고 해도, VARCHAR의 최대 길이가 255자를 넘어가면 데이터의 길이는 2바이트를 사용하여 저장하기 때문에, 65535가 아닌 65533까지만 선언이 가능하다.

-----------------------------------------

문제 7. utf8mb4 문자 인코딩 방식을 사용할 때, VARCHAR(16384) 로 선언이 가능하다. (O / X)

**정답** X
**해설** utf8mb4의 경우 4바이트가 필요한 이모지와 3바이트가 필요한 한글을 모두 커버하는 인코딩 방식이므로, 16383 까지만 선언이 가능하다.

문제 13. 기본키 중 인조키(surrogate key)가 무엇인가? (서술형)

**정답** 테이블의 행을 식별하기 위해 특별히 생성된, 일반적으로 숫자로 이루어진 고유하게 식별하는 키를 말한다.
**해설** 테이블 내에서 자동으로 생성되며, 일반적으로 아무런 실질적인 의미를 가지지 않는다.
