문제 10. OSI 7 Layer 와 TCP/IP 를 비교하시오. (서술형)

**정답** TCP/IP의 경우 4계층으로 나누는데, 그중 Application Layer을 OSI 7 Layer 에서는 Application Layer, Presentation Layer, Session Layer 의 세 계층으로 구분한다. 그리고 TCP/IP 의 Internet Layer 을 OSI 7 Layer 에서는 Network Layer 이라 부르며, TCP/IP 의 Network Access Layer 을 OSI 7 Layer 에서는 Data-Link Layer, Physical Layer 로 구분한다.
**해설** 구분을 좀 더 세분화 했을 뿐, 실질적으로 하는 역할에 대한 설명은 같다.

-----------------------------------------
